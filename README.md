# Heart Failure Mortality Prediction using Machine Learning

This project implements an end-to-end machine learning pipeline to predict patient mortality in heart failure cases using real-world clinical data. The workflow covers data preprocessing, exploratory analysis, class imbalance handling with SMOTE, and supervised classification models including Logistic Regression and Random Forest.

Model performance is evaluated using standard classification metrics such as accuracy, precision, recall, F1-score, confusion matrices, and ROC-AUC to assess predictive effectiveness and reliability.
